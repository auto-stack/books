# The Auto Programming Language

_by the Auto Language Team, with contributions from the Auto Community_

This book teaches you how to write programs in the Auto programming language.

Auto is a systems programming language designed for the AI era. It combines
zero-cost abstractions, memory safety without garbage collection, and a
concurrency model built on the Actor pattern. Auto code can be transpiled to
Rust (via `a2r`), and in the future, to other languages as well.

The code listings in this book show both Auto and Rust side by side. The Auto
version can be transpiled to the Rust version using the `a2r` tool.
