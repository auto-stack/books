**Goal:** 基于《A Byte of Python》原书结构，生成一本 Auto 语言（转译为 Python）的双语入门教程，放在 `byte-of-python/` 目录中。

## 1. 项目概述

基于 Swaroop C H 的《A Byte of Python》书籍结构，生成一本 Auto 语言的 Python 入门教程。Auto 的 a2p 转译器可以将 Auto 代码转译为 Python 3.10+ 代码，因此本书采用 Auto 代码 + Python 输出的双栏展示方式，与 rust/ 和 typescript/ 书的格式保持一致。

**核心策略：**
- 保留原书的章节结构和教学顺序
- 将原书的 Python 代码示例改写为 Auto 代码，同时展示 a2p 转译后的 Python 输出
- 每章提供英文和中文双语版本
- 代码示例使用 `<Listing>` 标签，内含 ````auto` 和 ````python` 双代码块
- listing 文件包含 `main.at` + `main.expected.py` + `pac.at`

**与原书的主要差异：**
- 安装章节改为安装 Auto 工具链 + a2p 转译器
- 所有 Python 代码示例改为 Auto 代码（展示转译后的 Python）
- 保留原书的教学风格（"How It Works" 解说、注意事项等）
- 删减/简化纯平台相关的安装细节（DOS Prompt 等）
- 附录中 FLOSS、Translation 等章节可简化或省略

## 3. Listing 格式规范

### 3.1 Markdown 中的 Listing 展示

使用 `<Listing>` 标签，包含 Auto 源码和 Python 输出：

```html
<Listing number="3-1" file-name="hello.auto" caption="Your first Auto program">

```auto
fn main() {
    print("Hello, World!")
}
```

```python
def main():
    print("Hello, World!")

if __name__ == "__main__":
    main()
```

</Listing>
```

### 3.2 Listing 目录中的文件

每个 listing 目录包含：
- `main.at` — Auto 源码
- `main.expected.py` — a2p 转译后的 Python 输出
- `pac.at` — Auto 包清单

**pac.at 格式（Python 目标）：**
```
name: "listing-03-01"
version: "0.1.0"
lang: "py"

app("listing-03-01") {}
```

**main.expected.py 格式：**
```python
# Auto-generated by a2p transpiler

def main():
    print("Hello, World!")

if __name__ == "__main__":
    main()
```

### 3.3 a2p 转译器能力与限制

**已支持：**
- `fn` 函数定义 → `def`
- `if`/`else if`/`else` → `if`/`elif`/`else`
- `for i in 0..10` → `for i in range(0, 10)`
- `is` (pattern matching) → `match`/`case` (Python 3.10+)
- `type` (struct) → `@dataclass` 或普通 `class`
- `enum` → `class Enum`
- f-string: `$var` → `{var}`
- `print()` → `print()`
- `.x` 成员访问 → `self.x`

**不支持 / 需注意：**
- `while` → Auto 中用 `for cond { ... }` 代替，转译为 `while cond:` (Python)
- lambda 函数（未实现）
- 列表推导式（未实现）
- `try`/`except`/`finally`/`with`（需验证支持）
- `import` Python 标准库（需验证支持）
- `global` 语句（需验证支持）

**应对策略：** 对于 a2p 不支持的特性，在 listing 中仍展示 Auto 代码，但 Python 输出部分手写等价的 Python 代码（标注为 "Equivalent Python"），而非依赖 a2p 自动转译。
