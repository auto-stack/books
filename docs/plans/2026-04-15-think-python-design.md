**Goal:** 基于《Think Python, 3rd Edition》原书结构，生成一本 Auto 语言（转译为 Python）的双语入门教程，放在 `think-python/` 目录中。

**Architecture:** 严格对齐原书 20 章（chap00-chap19）的结构，逐章将 Python 概念翻译为 Auto 概念。每章提供英文和中文两个版本（`.md` + `.cn.md`），代码示例使用 `<Listing>` 标签并包含 Auto 源码和 Python 转译输出。代码清单按章组织在 `listings/` 目录下。

## 1. 项目概述

基于 Allen B. Downey 的《Think Python, 3rd Edition》书籍结构，生成一本 Auto 语言的 Python 入门教程。Auto 的 a2p 转译器可以将 Auto 代码转译为 Python 3.10+ 代码，因此本书采用 Auto 代码 + Python 输出的双栏展示方式，与 `byte-of-python/` 等书的格式保持一致。

**原书特点：**
- Think Python 3e 以 Jupyter notebooks 为主要格式
- 每章结构一致：概念讲解 → 代码示例 → Debugging → Glossary → Exercises
- 大量使用图表（state diagrams、stack diagrams）—— Auto 版本中将简化为文字描述
- 使用自定义模块（thinkpython.py、diagram.py、Turtle.py）—— Auto 版本中不依赖这些
- 每章有 "Ask a virtual assistant" 练习提示
- 总计 81 个练习

**核心策略：**
- 保留原书的章节结构和教学顺序
- 将 Python 代码示例改写为 Auto 代码，同时展示 a2p 转译后的 Python 输出
- 每章提供英文和中文双语版本
- 代码示例使用 `<Listing>` 标签，内含 ````auto` 和 ````python` 双代码块
- listing 文件包含 `main.at` + `main.expected.py` + `pac.at`
- 简化/省略纯 Jupyter 特有的内容（cell magic、Colab 集成）
- 保留 Debugging 和 Glossary 小节
- 保留 Exercises 小节（但不包含答案）

**与原书的主要差异：**
- 所有 Python 代码示例改为 Auto 代码（展示转译后的 Python）
- 不使用 thinkpython.py 自定义模块
- 不使用 Turtle.py 图形模块（Auto 版本不适合）
- 不使用 diagram.py 图表生成（用文字描述替代）
- 不包含 Jupyter notebook 格式
- 安装章节改为安装 Auto 工具链

## 3. Listing 格式规范

与 `byte-of-python/` 完全一致：

```html
<Listing number="N-M" file-name="example.auto" caption="Description">

```auto
fn main() {
    print("Hello, World!")
}
```

```python
# Auto-generated by a2p transpiler

def main():
    print("Hello, World!")

if __name__ == "__main__":
    main()
```

</Listing>
```

**pac.at 格式：**
```
name: "listing-NN-MM"
version: "0.1.0"
lang: "py"

app("listing-NN-MM") {}
```
