# Preface

Auto is probably one of the few programming languages which is both simple and powerful. This is good for beginners as well as for experts, and more importantly, is fun to program with. This book aims to help you learn this wonderful language and show how to get things done quickly and painlessly — in effect 'The Anti-venom to your programming problems'.

## Who This Book Is For

This book serves as a guide or tutorial to the Auto programming language. It is mainly targeted at newbies. It is useful for experienced programmers as well.

The aim is that if all you know about computers is how to save text files, then you can learn Auto from this book. If you have previous programming experience, then you can also learn Auto from this book.

If you do have previous programming experience, you will be interested in the differences between Auto and your favorite programming language — I have highlighted many such differences. A little warning though, Auto is soon going to become your favorite programming language!

## Official Website

The official website of the Auto project is at <https://auto.dev> where you can learn more about the language, download the toolchain, and explore the documentation.
